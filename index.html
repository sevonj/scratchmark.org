# Markdown Editor for Writing

**Scratchmark** is a pleasant text editor for writing. It tries to give you everything you need and otherwise stay out of your way so you can just focus on the text. The app can be used for writing essays and making quick notes alike. Its file management is built around a folder structure that can handle large projects with lots of files. You can add any folder on your computer to the library, and move files around by dragging and dropping.

It uses Markdown, which means that ==all text formatting can be done by typing.==

It can display simple writing statistics, and has features, such as focus mode and fullscreen support to reduce distractions.

Markdown is an open and widely used markup language. Documents created with Scratchmark are more or less compatible with other similar apps. If you already have a "notebook" or a project from another app, you can just add the folder to Scratchmark's library. This also goes the other way: if you've had enough of Scratchmark, it's easy to switch to something else.

Available for Linux desktop.
