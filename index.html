# Markdown Editor for Writing

**Scratchmark** is a Markdown text editor designed to work well for writing longer texts and managing large projects. It's usable for writing essays and making quick notes alike. Or if you're mad enough, writing an entire novel. It's built around organizing your documents in a directory structure, with support for nested folders. You can add any folder to the library, and drag and drop items to move them around.

Scratchmark uses Markdown, which means that ==all text formatting can be done by typing.==

It can display simple writing statistics, and has features, such as focus mode and fullscreen support to reduce distractions.

Scratchmark is designed and developed by a power user, but that doesn't mean it's a hassle for a new user to pick it up. The editor is simple, intuitive, and pleasant. It tries to give you what you need and otherwise stay out of your way, so you can *just open the app and start writing.*

Markdown is a standard format, which means that documents created with Scratchmark are more or less compatible with other Markdown apps. If you already have a "notebook" or a project from another app, you can just add the folder to the Scratchmark's library. This also goes the other way: if you've had enough of Scratchmark, it's easy to switch to something else.

# A tool should serve its user

Scratchmark is a tool, and a tool should serve its user above anything else. Scratchmark does not collect information about you. It doesn't snoop around to find out which programs you have installed, or what kind of websites you visit. Your work will not end up in your advertisement profile, or be used for AI training. Scratchmark will never hold your files hostage, or deliberately make it harder to migrate them to a "competitor."

The project is free and open source. You can find the source code, available under GPLv3 license, from the Git repository.

Scratchmark just wants to help, no strings attached.
